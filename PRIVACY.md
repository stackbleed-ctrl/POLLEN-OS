# Privacy

Pollen is designed to be local-first.

## Principles
- data stays on-device by default
- no cloud dependency is required for local operation
- sensitive actions require explicit consent
- nearby cooperation should be opt-in and revocable

## Current repo state
This repository is an early tester build. Review all permissions and claims before publishing store copy.

## What to verify before public claims
- exact model download size
- exact RAM usage by device
- whether any dependency phones home
- what logs are written in debug builds
