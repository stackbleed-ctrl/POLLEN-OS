# Pollen early tester release

Pollen is an early build of a private AI operating layer for Android.

## What testers can try
- always-on foreground brain service
- notification intake
- local reasoning with fallback
- nearby device discovery scaffolding

## What feedback we want most
- usefulness
- trust
- battery impact
- device compatibility
- permission comfort level

## Known limitations
- nearby delegation is early
- local model integration is still being hardened
- not all autonomous actions are wired
- this is not a custom Android OS
