# Feedback guide

Please focus feedback on:

## 1. Real usefulness
- Which task felt genuinely helpful?
- Which task felt creepy, noisy, or unnecessary?

## 2. Permission trust
- Which permissions felt acceptable?
- Which permissions need more explanation or better opt-in flow?

## 3. Reliability
- Did the service stay alive?
- Did actions happen when expected?
- Did anything fail silently?

## 4. Performance
- Was the phone warm?
- Was battery drain noticeable?
- Did inference feel usable?

## 5. Nearby cooperation
- Did multi-device discovery work?
- Was delegation understandable?
- Did trust/consent feel clear?

Please include:
- device model
- Android version
- exact steps
- screenshots or short video if possible
